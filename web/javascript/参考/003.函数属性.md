## eval()

对表示为字符串的JavaScript代码进行计算。

```
eval(string)
```

## uneval()

已废弃

创建Object的源代码的字符串表示形式。

```
uneval(object)
```

## escape()

已废弃

对字符串进行编码。

```
escape(str)
```

## unescape()

已废弃

对字符串进行解码，其中十六进制转义序列替换为它表示的字符。



```
unescape(str)
```
